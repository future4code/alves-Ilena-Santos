Blackjack

Inspirado no jogo de cartas Blackjack, essa versão é jogada no navegador de web. A disputa é feita entre o usuário e o computador. As cartas são sorteadas em modo aleatório e a pontuação das cartas somadas em cada rodada. Ganha o jogador que tem pontuação menor que 21. Para o desenvolvimento do projeto foram utilizados o editor de código VScode e a linguagem de programação JavaScript.
